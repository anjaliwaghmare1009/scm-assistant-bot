# Architecture Overview

The SCM Assistant is an AI-powered chatbot built using a Retrieval-Augmented Generation (RAG) architecture to answer supplier and supply-chain-related queries.

## Workflow

1. Dataset Ingestion
2. Supplier-Level Document Generation using Python Preprocessing
3. OpenAI Embedding Generation
4. Storage in Pinecone Vector Database
5. Similarity-Based Retrieval
6. Context-Aware Answer Generation using GPT-4o-mini

---

# Data Preprocessing

## Initial Challenge

The original dataset contained Purchase Order (PO)-level records.

Since every purchase order was stored separately, a single supplier appeared multiple times in the dataset. This caused several issues:

* Duplicate supplier retrieval
* Inconsistent supplier counts
* Different answers for similar queries
* Difficulty in answering supplier-level analytical questions

---

## Solution

A Python preprocessing script was developed to aggregate multiple purchase-order records belonging to the same supplier and generate consolidated supplier-level documents.

This significantly improved retrieval quality and enabled accurate supplier-level analysis.

---

# Supporting Files

## supplier_aggregation_script.py

This script aggregates all purchase-order records belonging to the same Supplier_ID and generates consolidated supplier profiles containing:

* Supplier Metrics
* Risk Information
* Compliance Scores
* Certifications
* Disruption Flags
* Sustainability Data

---

## supplier_profiles.txt

This file contains supplier-level documents generated by the preprocessing script.

These documents are indexed into Pinecone and act as the primary knowledge base for the RAG chatbot.

# Sample Questions

### Q1. Which Tier-3 suppliers have an active disruption flag?

The chatbot identifies high-risk suppliers with active disruption flags and provides response levels according to policy guidelines.

### Q2. Which suppliers qualify for the annual Volume Rebate Program?

The system retrieves eligible suppliers based on performance metrics, defect rates, and sustainability scores.

### Q3. Which region has the highest total PO value?

The chatbot performs supplier-level analysis and determines whether concentration limits are breached.

### Q4. Which suppliers are on the Supplier Watch List (SWL)?

The system identifies suppliers with low compliance scores and explains applicable restrictions.

### Q5. Which product category has the highest average defect rate?

The chatbot analyzes category-wise defect rates and checks compliance with threshold policies.
